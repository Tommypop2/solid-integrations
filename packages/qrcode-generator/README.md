A sample primitive that is made up for templating with the following options:

`createPrimitiveTemplate` - Provides a getter and setter for the primitive.

## Installation

```bash
npm install @solid-integrations/qrcode-generator
# or
yarn add @solid-integrations/qrcode-generator
# or
pnpm add @solid-integrations/qrcode-generator
```

## How to use it

```ts
const [value, setValue] = createPrimitiveTemplate(false);
```
